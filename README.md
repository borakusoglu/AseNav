•  Run the AseNav setup.exe file.

•  Launch Aseprite and go to File > Scripts > Open Scripts Folder.

•  Place the AseNav.lua file into the opened folder.

•  Go back to Aseprite and press F5 to refresh the scripts.

•  Go to File > Scripts, find AseNav, and click it.

•  Allow any permissions the script requests.

•  You’re all set to start using AseNav!
